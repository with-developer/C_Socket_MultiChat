# C_Socket_MultiChat<br>
Tcp/Ip C Socket Multi Chatting<br>

C언어 기반 소켓 멀티 채팅 프로그램입니다.<br>

기능<br>
\help - 도움말 표시 <br>
\users - 사용자 목록 출력<br>
\quit - 클라이언트 종료<br>
\private - 귓속말<br>
\privates - 내게 귓속말을 보낸 사용자<br>
\admin - 관리자 권한 부여<br>
\ban - 사용자 정지(admin)<br>
\kick - 사용자 추방(admin)<br>
\nick - 사용자 이름 변경(admin)<br>
\shutdown - 서버 종료(admin)<br>
